How it works

Resolve grievances in 4 steps

From submission to resolution — every step is transparent and trackable.

01
📝

Submit

Fill the 2-step form with your category, sub-category, priority, and description. Anonymous mode available.

02
🎫

Get a Token

Receive a unique token ID instantly after submission. Use it anytime to check your grievance status.

03
⚙️

Admin Reviews

Admin assigns the grievance to an officer. If unresolved, it escalates automatically up to 3 levels.

04
✅

Resolution

A resolution note is added. Track the full timeline, export history as CSV, and view audit trails.

Platform Features

Everything you need

12+ modules built for students, admins, and officers — all in one portal.

📋

Multi-step Grievance Form

Category, sub-category, priority level, description — with an anonymous submission toggle.

🔍

Token-based Status Tracker

Paste your unique token ID to get real-time status updates and a full resolution timeline.

⚖️

3-Level Escalation

Grievances auto-escalate if unresolved. Admins can manually escalate with a single click.

👮

Officer Assignment

Admins assign grievances to specific officers who are responsible for resolution.

📊

Admin Analytics Dashboard

Bar charts, resolution rates, and category-wise breakdowns for system-wide insights.

📤

CSV Export

Students and admins can export grievance history as CSV for records and reporting.

🔐

Role-based Auth

Separate dashboards and access controls for Student, Admin, and Officer roles.

🕵️

Anonymous Mode

Students can hide their identity while still getting their grievance tracked and resolved.

📜

Filterable History

View all past grievances filtered by status, category, or priority with sort options.

User Roles

Built for everyone

Three distinct roles, each with a tailored dashboard and access level.

🎓

Student

Default role
  • Submit grievances with category & priority
  • Track status using token ID
  • View personal history & timeline
  • Export own grievances as CSV
  • Toggle anonymous mode
🛡️

Admin

Full access
  • View & manage all grievances
  • Assign grievances to officers
  • Escalate to Level 2 or Level 3
  • Add resolution notes
  • Access analytics dashboard
  • Export all records as CSV
⚙️

Officer

Assigned handler
  • View assigned grievances
  • Update status and add notes
  • Access admin panel view
  • Communicate resolution details
