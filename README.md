# Student Grievance Redressal Portal

## About Project

This project is a web application developed to help students submit and track their grievances online.  
Students can register complaints, view complaint status, and communicate easily with the administration.  
The system provides a simple and user-friendly interface for grievance management.

## Features

- Student complaint submission
- Complaint status tracking
- User-friendly dashboard
- Secure login system
- Admin grievance management

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB

## How to Run

1. Make sure you have Node.js installed (https://nodejs.org)

2. Open a terminal in this folder

3. Run:

```bash
npm install
npm run dev
